  • IHMC CatVar: A large database of clusters of uninflected English words and their categorial variants (Open Software License version 1.1)
  • IHMC LCS: Manually created large database of lexical conceptual structures for English (GNU General Public License)
  • Original LCS classes based verbs list: The list of verbs identified based on original LCS database used for ask (and framing) detection
  • LCS+ classes based verbs list: The list of verbs identified based on enhanced LCS database tailored to Panacea used for ask (and framing) detection
  • Thesaurus based verbs list: The list of verbs identified based on Thesaurus database used for ask (and framing) detection
